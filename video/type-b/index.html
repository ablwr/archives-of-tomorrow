Fun facts
  • This format began to replace 2" Quad and 1" Type A, but it was later largely replaced by 1" Type C for use in broadcast television
  • This format segments each field to 5 (NTSC) or 6 (PAL) helical scan tracks per field, instead of recording one field per track
  • This format can be distinguished from the other 1" types by the way the tape wound, with the oxide facing out instead of in; Type C tape would face in and Type A could be either if created before 1975
