Fun facts
  • This was the first HD video format available in a Betacam-sized cassette
  • HDCAM cassettes were the same size as original Betacam but had a distinctive orange lid (and HDCAM SR cassettes were black with a cyan lid)
  • This format's codec used non-square pixels
  • This format's codec used non-square pixels, so the recorded 1440x1080 content was then upsampled to 1920x1080 on playback
  • Some HDCAM VTRs play back older Betacam variants
