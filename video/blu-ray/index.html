Fun facts
  • This format's name refers to the "blue" (violet) laser used to read the disc, which is stronger than the (longer wavelength) red laser used to make DVDs
  • While most discs are region-free, the Blu-ray Disc region coding scheme divides the world into three regions, labeled A, B and C
  • These discs can have up to four layers, with each layer offering more potential capacity
