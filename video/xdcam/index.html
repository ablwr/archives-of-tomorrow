Fun facts
  • This format uses random access solid-state memory media cards to store digital video
  • This format was mostly used for high-end professional camera work
  • This format included four different products, XDCAM SD, XDCAM HD, XDCAM EX and XDCAM HD422, which used different encoders, frame size, container types, media, and were marketed for different purposes and budgets
