Format
analog

Olympus
(Max) Capacity
30 minutes (per side)
Size
5 × 3.5 × 5 cm
Fun facts
  • This format has the same width of magnetic tape as the Compact Cassette but in a container roughly one quarter the size
  • This format was commonly used in dictation machines or telephone answering machines, but also computer data storage
  • This format was slightly more narrow (by 5mm) than the Mini-cassette
